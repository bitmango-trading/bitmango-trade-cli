## Core Principles

-   **Unified Interface:** The same command works across all supported exchanges.
-   **Exchange Agnostic:** Switch between exchanges with a single flag.
-   **Professional Logic:** Built-in support for advanced execution strategies.
-   **Bot Ready:** High-performance JSON output mode for programmatic use.
-   **Visual Feedback:** Professional boxed UI with real-time progress indicators.
